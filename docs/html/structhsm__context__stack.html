Detailed Description

Per state machine instance manager of context data.

note: the most effienct implementation depends on the sparsity of the stack if it's very sparse: a list of (context,depth) tuples might be best if it's very full: an array of [context,...] with blank elements(*) might be best flagging is somewhere in between, though, all methods are probably fine for most people.
