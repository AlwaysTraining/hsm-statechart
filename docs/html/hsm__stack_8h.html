Add a new context to the stack.

Parameters:
stackCan be NULL.
contextThe new context. NULL is legal, but not recommended.
Note:
The current implementation uses a linked list of pointers, so handling NULL is necessary or the list would break. currently, therefore, a context of NULL means "reuse the parent context", but it's possible this behavior may change in the future.
